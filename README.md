# 🏖️ Explore o Rio - Guia Turístico do Rio de Janeiro

Um site responsivo e moderno apresentando os principais pontos turísticos do Rio de Janeiro, com informações, curiosidades e galerias de imagens.

## 🌟 Recursos

- **Homepage atraente** com cards interativos dos destinos
- **Páginas detalhadas** para cada ponto turístico (Cristo Redentor, Copacabana, Pão de Açúcar)
- **Design responsivo** que se adapta a diferentes tamanhos de tela
- **Galeria de imagens** para cada destino
- **Informações e curiosidades** sobre cada lugar
- **Navegação intuitiva** entre páginas

## 🎨 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e efeitos
- **Google Fonts** - Font "Poppins" para melhor tipografia

## 🚀 Como Usar

1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` em seu navegador
3. Explore os destinos clicando nos cards da página inicial
